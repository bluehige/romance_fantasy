# 문체 추가 지침

## 1. 메타발언 금지
- `2화에서는 이렇게 행동했었다` 같은 식의 화수 메타 언급 금지
- `이전 화에서 설명했듯` 같은 작품 바깥 설명 금지
- 이전 사건은 현재 인식, 회상, 대화, 반응으로만 드러낼 것

## 2. 어려운 단어 과용 금지
- 평소 잘 쓰지 않는 과한 한자어, 번역투, 현학적 단어 남발 금지
- 쉬운 말로 자연스럽게 쓸 수 있으면 쉬운 표현 우선
- 캐릭터 입에서 나올 법한 표현인지 확인할 것
