# 집필 에이전트 역할 요약

- story-planner: 로맨스/직무갈등/정치 떡밥 설계
- scene-designer: 티키타카, 감정선, 권력 충돌을 씬으로 분해
- draft-writer: 초고 작성
- research-qa: 세계관/정치/계약 구조 검수
- character-qa: 감정선/여주 능동성/남주 입체감 검수
- revision-editor: 설명 과다, 감정선 급가속, 구조 혼란 조정
- publication-editor: 로코 호흡과 여운 살린 공개본 정리
- final-reviewer: 실제 파일 존재 및 단계 완결성 검수
