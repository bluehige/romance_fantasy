# 황인불퇴코덱스버전

`황실 인사팀에서 불멸기사를 퇴사시키는 법`을 사용자와 Codex가 같이 쓰기 위한 협업 구조다.

이 폴더는 자동 에이전트 엔진용이 아니라 실제 집필 흐름용이다.

읽는 순서:

1. `00_시작/00_여기부터.md`
2. `01_캐논/작품개요.yaml`
3. `01_캐논/세계관/*`
4. `01_캐논/인물/*`
5. `01_캐논/관계/*`
6. `02_플롯/시즌01/*`
7. `03_원고/시즌01/황인불퇴_시즌01_에피01_직원번호 0001.md`

원칙:

- 캐논 수정은 `05_변경관리/`를 먼저 거친다.
- 플롯 문서는 집필의 기준이지만 원고보다 우선하지 않는다.
- 원고에서 새 설정이 필요하면 먼저 변경 제안을 남긴다.
- 검수는 원고를 직접 고치지 않고 `04_검수/`에 티켓으로 남긴다.
